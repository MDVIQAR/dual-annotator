# Dual Annotator

A unified annotation tool for YOLO (object detection) and U-Net (segmentation) datasets.

## Features
- YOLO mode: Bounding box annotation with copy-paste-resize
- U-Net mode: Segmentation masks with solid and ring shapes
- Multi-class support with color coding
- Export to YOLO TXT format

## Installation
1. Create virtual environment: `python -m venv venv`
2. Activate: `venv\Scripts\activate`
3. Install: `pip install -r requirements.txt`
4. Run: `python main.py`
